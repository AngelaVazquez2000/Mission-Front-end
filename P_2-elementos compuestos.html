Explicación de elementos compuestos

Un imput de texto nos permite crear una seccion en la que se podemos meter información este puede tener diferentes caracteres.







Input De Texto


El imput de correo funciona para dar a interpretar al codigo que estamos trabajando con textos de correo electronico.



El imput de correo solamente admite valores númericos,




El imput de Password nos sirve para poner el caracter "*" característico de las contraseñas.




El imput de telefono son caracteres numericos, sin embargo, se pueden modificar la cantidad de caracteres con "maxlenfth".







Input de selección multiple


Funciona de manera que podemos seleccionar diferentes opciones dentro de un listado.
Por ejemplo: selecciona cuales son tus sabores de helados favoritos.






Input de selección única


Funciona de manera que solo se puede elegir una opción dentro de un listado. Por ejemplo: selecciona la respuesta correcta de ¿Cuantos lados tiene un circulo?





Select

Funciona para seleccionar respuestas de un listado.
Por ejemplo: selecciona el mes del año en el que naciste.

Select con Input

Es un select con input mas avanzado con la opción de entrada de texto.




Input de Fecha


Se usa para establecer fechas.

La siguiente para introducir fecha a traves de un calendario.


La siguiente para introducir fecha con hora.


La siguiente para introducir fecha pero por semana.


La siguiente para introducir tiempo en horas.


La siguiente para introducir fecha solamente con mes y año.



Input de archivo


Se utiliza para ingresar archivos.



Input de Color


Se utiliza para seleccionar un color.



Input de Rango

Se utiliza para para selecionar rangos.



Input de botón

Para poder insertar botones con diferentes acciones.

Input submit= enviar información.
